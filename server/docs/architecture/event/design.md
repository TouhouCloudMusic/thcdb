# Event (活动) 模块

> **实现状态**: ⚠️ 部分完成 (仅查询) | [查看路线图](../ROADMAP.md)

音乐活动实体，代表音乐会、音乐节、专辑发布会、录音会话和其他音乐相关活动。

## 基本信息

- **Name / 名称**: 活动的主要名称
- **Short Description / 简短描述**: 活动的简要说明
- **Description / 详细描述**: 活动的完整描述
- **StartDate / EndDate / 开始/结束日期**: 活动的时间范围，包含精度信息 (DatePrecision)

日期精度见 [DatePrecision](../shared-types.md#dateprecision)。

## 相关实体

### 核心关联

- **发行版本**: 与活动关联的专辑、单曲或其他发行版本
- **替代名称**: 同一活动的不同名称或标题

### 发行版本连接

活动通过各种关系连接到发行版本：

- **发布活动**: 专辑发行派对或宣传活动
- **现场录音**: 产生现场专辑的音乐会

## 替代名称

活动可以通过 `event_alternative_name` 实体拥有多个名称：

- **缩写**: "SMF" 代表 "夏季音乐节"
- **俗称**: 正式活动名称的"大型演出"
- **历史名称**: 随时间变化的名称
- **本地化名称**: 不同语言的名称

## 历史追踪

相关历史表：
- `event_history` - 活动基本信息变更历史
- `event_alternative_name_history` - 活动替代名称历史

## API 端点

| 端点 | 方法 | 状态 | 说明 |
|------|------|------|------|
| `/event` | GET | ✅ | 查询活动列表 |
| `/event/{id}` | GET | ✅ | 获取活动详情 |
| `/event` | POST | ❓ | 创建活动 (需确认) |
| `/event/{id}` | PUT | ❓ | 更新活动 (需确认) |

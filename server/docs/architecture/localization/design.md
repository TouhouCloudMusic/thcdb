# Localization (本地化) 模块

> **实现状态**: ✅ 数据层已完成 | [查看路线图](../ROADMAP.md)

本地化系统为 THCDB 中的所有实体提供全面的多语言支持，实现国际化访问和文化适应。

## 系统概述

THCDB 中，每个主要实体都有自己的本地化名称表。

## 架构

### Language 实体

语言注册表包含：

- **代码**: ISO 语言代码（如 "en"、"ja"、"zh"）
- **名称**: 语言的显示名称

### 本地化名称

每个可本地化的实体都有自己的专用本地化表：

- **Artist**: 艺人名称本地化
- **Release**: 发行版本标题本地化
- **Song**: 歌曲标题本地化
- **Event**: 活动名称本地化
- **Label**: 厂牌名称本地化
- **Tag**: 标签名称本地化

### 本地化记录

每条本地化记录包含：

- **实体引用**: 关联的实体
- **语言**: 该名称的语言
- **名称/标题**: 本地化后的文本
- **是否原始**: 标识是否为原始语言版本

## 关键功能

### 原始语言指定

每个本地化条目都可以标记为原始语言：

- **is_original**: 指示源语言的布尔标志
- **源追踪**: 识别哪个版本是原始版本

### 多语言搜索

搜索功能跨越所有本地化版本：

- **跨语言搜索**: 无论语言如何都能找到内容
- **罗马化支持**: 用罗马字搜索日语内容
- **文字转换**: 处理不同的书写系统
- **模糊匹配**: 考虑音译变体

## 历史追踪

所有本地化表都有对应的历史表：

- `artist_localized_name_history`
- `release_localized_title_history`
- `song_localized_title_history`
- `label_localized_name_history`
- `tag_alternative_name_history`
- `event_alternative_name_history`

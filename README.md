# Touhou Cloud DB（THCDB）

Touhou Cloud DB（THCDB）是一个面向东方同人音乐的数据库项目。

项目定位与功能参考了 Rate Your Music、VocaDB、Discogs 等平台。

## 目前状态

项目仍在建设中，模块进度见 [docs/architecture/README.md](docs/architecture/README.md)，近期计划见 [docs/architecture/ROADMAP.md](docs/architecture/ROADMAP.md)。
